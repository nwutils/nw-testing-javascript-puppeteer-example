# NW.js Testing Examples

## Getting Started

### JavaScript

1. `npm i` to install Node modules
1. `npm t` to run tests

### Python

1. `npm i` to install Node modules
1. `python3 -m pipenv install` to install Python dependencies
1. `python3 -m pytest ./py/selenium/builder/test.py` to run tests

## Languages

- Java
- JavaScript
- Python
